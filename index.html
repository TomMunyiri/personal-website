This is an application for showing realtime reports and analytics from the SimbaPOS Desktop app. It connects to AWS from where the reports are generated. I created the REST Api using PHP and the Android app using Java and Kotlin

Kikao

Android, PHP, MYSQL

Kikao is an app that allows you to list and search for rental houses. It allows you to locate a house on Google Maps. You don't have to create a user account to use Kikao to search rentals. A user account is only needed when listing. Created using PHP for backend and Java(Android). *It is not available on Play Store right now but I can send you the Demo app to have a look*
